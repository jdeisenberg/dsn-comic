Where do I find the files?

  • You can find scans of the original pencil art in the pencilscans directory.
  • The colored and inked files are in the colored-inked directory.
  • The files used to create the comic are in the source-files directory.
    • The files ending in .svg are in SVG format, and you can use Inkscape to edit them.
    • The font for the comic is in the file Driftjunky13_alpha.ttf
    • I used the script make_pdf.sh to create the PDF file. It uses Inkscape to convert each SVG file to PDF, and then uses pdftk (the PDF toolkit) to combine the PDF files into one large file.
