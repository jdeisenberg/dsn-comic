Where do I find the files?

Get the files at git://github.com/jdeisenberg/dsn-comic.git (If you don’t have git, you can download it from http://git-scm.com/).

Once you have gotten the files, you’ll find these subdirectories and files.

  • Scans of the original pencil art are in the pencilscans directory.
  • The colored/inked illustrations without any text on them are in the base-jpg directory.
  • The SVG files that have the text and embedded images are in directories for each localization, such as en-US or de-DE.
  • The fonts used in the comic are in the fonts directory.
  • The make_pdf.sh file builds the main PDF. It uses Inkscape to convert each SVG file to PDF, and then uses pdftk (the PDF toolkit) to combine the PDF files into one large file.
  • The dsn-comic.po file contains strings extracted from the SVG files. I’m not sure how easy it will be to translate from those strings, given that they don’t fit into some nice rectangular window.
